Introduction to HTML

HTML (Hyper Text Markup Language) is a standard language used to create and design webpages. It is called a "Markup Language" because it uses tags to define elements like heading, paragraph, images, ordered list etc.. and to "markup" content to tell a web browser how to display it.

Introduction to CSS

CSS ( Cascading Style Sheets) is used to style webpages. Its control how HTML elements look like their colors, fonts,sizes,and layout. CSS helps website to use to look nice and attract users and make them easy to use.

Introduction to Javascript

Javascript is a programming language that make websites to do things.It allows to let buttons work, forms, react and real time updates without reloading the page.
